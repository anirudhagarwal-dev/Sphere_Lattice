## About

Sphere packing is a classic mathematical and geometrical problem focused on arranging spheres in the most efficient way possible. **Sphere_Lattice** brings this concept to life with a smooth, real-time visualization that allows users to observe lattice-like structures forming dynamically.

This project is ideal for:
- Learning and visualizing geometric concepts  
- Interactive demonstrations  
- Creative coding and experimentation
